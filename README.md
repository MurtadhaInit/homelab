# 🏡 Homelab

## Overview

This repository shows my entire homelab (i.e. home servers) as defined and configured using various IaC and automation tools.

## Motivation

The rationale behind creating a homelab was primarily as hobby and because of an interest in self-hosting; for privacy, for data ownership, and for fun. The second motivating factor is for learning and experimenting with different and technologies, including emerging ones, to 1. Improve my workflows and making my local deployments more robust and resilient, 2. I generally love learning, and 3. It serves as a showcase for my skills.
